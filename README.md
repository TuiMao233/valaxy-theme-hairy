> Hairy theme based on valley.

## Usage

### create valaxy

> Use [pnpm](https://pnpm.io/), because we need its workspace.

```bash
pnpm create valaxy
```

### install theme

```bash
cd project
pnpm add valaxy-theme-hairy
```

edit valaxy.config

```json
{
  "theme": "hairy"
}
```
